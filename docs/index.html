Tolkeassistent til hverdagsbrug

En open source platform til oversættelse med talegenkendelse, AI-drevet oversættelse og naturlig stemmesyntese.

KK AI Translator blev bygget af Kalundborg Kommune (Danmark) for at hjælpe sagsbehandlere med at kommunikere med borgere, der taler forskellige sprog. Den er nu open source og gratis at bruge.

Appen kører i din browser. Tal ind i din mikrofon, og den transskriberer, oversætter og læser oversættelsen højt - i begge retninger.

Funktioner

Tale-til-Tekst

Transskriber talt lyd ved hjælp af Azure Speech Services eller Whisper. Understøtter 450+ sprog.

AI-oversættelse

Oversæt mellem sprog ved hjælp af GPT-4o-mini eller andre LLM'er med naturlige formuleringer.

Tekst-til-Tale

Hør oversættelser læst højt med naturligt klingende stemmer via Azure TTS.

Konfigurerbare Modeller

Vælg hvilken AI-model der håndterer transskription, oversættelse og sammenfatninger for hvert sprog.

Sessionhistorik

Spør samtaler med fulde transskriptioner og AI-genererede sammenfatninger på begge sprog.

Nem Opsætning

Docker Compose får dig i gang på få minutter. Tilføj bare dine Azure API-nøgler.

Kom i gang

Hvad du skal bruge

  • Docker Desktop - Download her (gratis)
  • Azure-konto med Speech Services og OpenAI - Gratis niveau tilgængeligt
  • Cirka 10 minutter

Download koden

Klon repository'et til din computer:

git clone https://github.com/kalundborgkommune/KK-AI-Translator-V1-OS.git
cd KK-AI-Translator-V1-OS

Tilføj dine API-nøgler

Kopiér eksempel-miljøfilen og tilføj dine Azure-legitimationsoplysninger:

cp .env.example .env
# Åbn .env i en hvilken som helst teksteditor og udfyld:
# AZURE_SPEECH_KEY=din_nøgle_her
# AZURE_OPENAI_KEY=din_nøgle_her

Se SETUP.md for detaljerede instruktioner om at få Azure-nøgler.

Start appen

Kør denne kommando og vent på at den er færdig:

docker compose up -d

Første kørsel tager et par minutter at downloade alt.

Åbn i browser

Gå til http://localhost:8080 og begynd at oversætte!

Første gang? Besøg Indstillinger og klik på "Seed Languages" for at indlæse alle tilgængelige sprog.

Deploy til Azure

Klar til at deploye til produktion? Der er to tilgange afhængigt af dine behov:

Option A: Azure VM

Simpel deployment ved hjælp af Docker Compose på en Linux VM. Perfekt til at komme hurtigt i gang med fuld kontrol.

Option B: Azure Web App

Administreret serverless deployment med Azure Container Registry og Azure MySQL. Auto-scaling og høj tilgængelighed.

Hurtig Start: VM + Docker Compose

Den enkleste tilgang - kør alt på en enkelt Azure VM:

# Opret VM og SSH ind
az vm create --resource-group translator-rg --name translator-vm --image Ubuntu2204 --size Standard_B2s

# På VM'en: installer Docker og klon repo
curl -fsSL https://get.docker.com | sh
git clone https://github.com/kalundborgkommune/KK-AI-Translator-V1-OS.git
cd KK-AI-Translator-V1-OS && cp .env.example .env

# Konfigurer .env med dine Azure-nøgler, start derefter
docker compose up -d

Hurtig Start: Azure Web App (Serverless)

Til produktion med administrerede tjenester:

# Byg og push til Azure Container Registry
make build-full
make push-full

# Deploy til Azure Web App
make deploy-full

Se SETUP.md for detaljerede trin-for-trin instruktioner inklusive Azure MySQL opsætning, SSL-konfiguration og kontinuerlig deployment.

Konfiguration

Sprogindstillinger

Indstillingssiden (/settings) lader dig konfigurere hvert sprog:

  • Aktivér/deaktiver - Vælg hvilke sprog der vises i oversætteren
  • Stemmevalg - Vælg fra flere stemmeindstillinger per sprog
  • AI-modeller - Vælg hvilken model der håndterer transskription, oversættelse og sammenfatninger

Understøttede Modeller

Ud af boksen understøtter appen:

  • Azure Speech - Microsofts tale-til-tekst (anbefalet)
  • Whisper via Promte - Alternativ transskriptionsendepunkt
  • GPT-4o-mini - Hurtig, overkommelig oversættelse

At tilføje nye modelintegrationer er ligetil - se koden i python-be/src/services/.
